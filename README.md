# My Cellar

A personal wine cellar management app. Track your wine collection, maintain a complete history of additions and consumption, and rate wines over time. Built as a mobile-first static web app with Svelte 5, Vite, and Tailwind CSS.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- npm

## Getting started

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

The app will be available at `http://localhost:5173/my-cellar/`.

## Available scripts

| Command          | Description                          |
|------------------|--------------------------------------|
| `npm run dev`    | Start the development server with HMR |
| `npm run build`  | Build for production                 |
| `npm run preview`| Preview the production build locally |
| `npm run check`  | Run TypeScript and Svelte type checks |
| `npm test`       | Run tests once                       |
| `npm run test:watch` | Run tests in watch mode          |
